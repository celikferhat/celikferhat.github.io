Windows'un Tarihi

İşletim sistemi, bilgisayarda çalışan, donanım kaynaklarını yöneten ve çeşitli uygulama yazılımları için yaygın servisleri sağlayan bir yazılımlar bütünüdür. İşletim sistemi, uygulama kodları genellikle direkt donanım tarafından yürütülmesine rağmen, girdi-çıktı, bellek atama gibi donanım fonksiyonları için uygulama programları ve bilgisayar donanımı arasında aracılık görevi yapar. Windows da bu görevi üstlenen en eski işletim sistemlerinden biridir.Her defasında kendini yenileyen,yeni sürümler çıkaran, kapalı kaynak kodlu bir işletim sistemidir. Şimdi Windows'un sürümlerini inceleyelim

Windows 1.0

Microsoft‘un ilk stabil Windows sürümü olan Windows 1.0, satış tarafında pek başarılı olamamıştır. İlk Windows olduğu için yeniliklerinden pek fazla bahsedemediğimiz Windows 1.0, MS-DOS üzerine geliştirilen 16 bitlik bir işletim sistemidir. Windows 1.0 ile birlikte ilk kez Notepad, Paint, Writer gibi temel Windows yazılımları ortaya çıkmıştır.

Windows 2.0

Windows 2.0, Windows 1.0 üzerine geliştirilmiş ve Windows 1.1 olarak da değerlendirilebilecek bir Windows sürümü olmuştur. İlk kez Ofis ailesinin Word ve Excel üyeleri, Windows 2.0 ile birlikte duyurulmuş, ancak Mac OS X benzerliği nedeniyle Microsoft ve HP, Apple tarafından açılan davalar ile boğuşmuştur.

Windows 3.0

İlk kez oyunların geldiği Windows 3.0, Solitaire ve Minesweeper gibi oyunların çıktığı işletim sistemi olmuştur. Windows işletim sistemleri arasında ilk kez büyük başarı yakalayan sürüm olan Windows 3.0, kullanıcı arayüzünde önemli iyileştirmelerle gelmişti.

Windows 95

1995 yılında duyurulması nedeniyle 95 adını kullanan Windows 95, ilk 32 bit Windows sürümüdür. Başlat menüsü ve Internet Explorer‘ın çıkış yaptığı işletim sistemi olan Windows 95 ile aynı zamanda görev çubuğu da kullanıma sunulmuştur.

Windows 98

Windows 95 gibi duyurulduğu yıla göre adlandırılan Windows 98, geliştirilen internet hizmetleri, hızlı başlat ve harici aygıtlar ile hızlı bağlantı özelliği ile dikkat çekmiştir. Outlook yazılımının da çıkış noktası olan Windows 98, Microsoft WebTV yazılımı ile TV izleme olanağı sunmuştur.

Windows ME

Windows‘un 2000 yılında duyurulan ve son kez duyurulduğu yıla göre adlandırılan sürümü olan Windows Milennium Edition, adını Milenyum döneminden almıştır. Windows 98‘deki tüm sorunların onarıldığı Windows ME, Windows Live Movie Maker ve Windows Media Player yazılımlarına ev sahipliği yapmıştır. Ayrıca Windows ME ile birlikte sistem geri yükleme özelliği duyurulmuştur.

Windows XP

Microsoft tarafından kişisel bilgisayarların yanı sıra sunucular için de geliştirilen Windows XP, Microsoft‘un en iyi işletim sistemlerinden biri olarak gösterilebilir. Korsanın büyük oranda engellendiği Windows XP, çok iyi satışlar elde etmiştir. ClearType, Uzaktan Erişim, Kullanıcı Hesapları ve Internet Explorer 6 yazılımları Windows XP ile birlikte gelmiştir.

Windows Vista

Microsoft‘un en başarısız işletim sistemlerinden biri olan Windows Vista, Windows Aero adındaki saydam arayüz özelliği ile dikkat çekmiştir. Windows kenar çubuğunun da ilk kez geldiği Windows Vista, XP’de çok stabil çalışan pek çok seyi bozması ile ilk kez duyurulduğunda hiç sevilmemiş, Service Pack 1 ve 2 güncellemeleri ile yaşanan sorunlar düzelmesine karşın, kullanıcılar Vista‘yı tercih etmemiştir.

Windows 7

Windows XP sonrasında en çok sevilen Windows 7 ile birlikte, ilk kez geliştirici sürümleri denenmiş, bu şekilde güncelleme öncesinde tüm sorunlardan kurtulunarak 2009 yılında Windows 7 yayınlanmıştır. Esasında yedinci temel Windows sürümü olması nedeniyle Windows 7 ismini alan işletim sistemi, geliştirilen görev çubuğu ve arayüz özellikleri ile ön plana çıkmıştır.

Windows 8

Windows 7 sonrasında Metro arayüzü ile dikkat çekmeye çalışan Windows 8, uygulama mağazası ile yeni bir yapıya kapılarını açmaya çalışsa da, kullanıcıların beğenisini kazanamamış. Hatta başlat menüsünün gitmesi, kullanıcıların Windows 8‘i özellikle tercih etmemesine neden olmuştur.

Windows 8.1

Windows 8‘de yaşanan sorunların giderildiği ve Windows 7 kadar işlevli olmasa da bir başlat menüsünün getirildiği Windows 8.1, Windows 8 ile kıyaslandığında çok daha fazla beğenilmiş, ancak yine de Windows 7 kadar başarılı olamamıştır.

Windows 10

En son Windows sürümü olan Windows 10, başlat menüsünün geri gelip metro arayüzü ile birleşmesi, Cortana, Microsoft Edge gibi pek çok yenilikle gelmiş ve yapılan araştırmalarda kullanıcılardan tam puan almıştır. Windows 10 Yıldönümü güncellemesi ile sistemdeki birçok hata giderildiği gibi yeni özellikler de kullanıcılara sunuldu. Şu an en son sürüm olan windows 10'u kullanmanızı tavsiye ediyoruz.Windows 10 'un birden fazla sürümü mevcut doğru sürümü seebilmeniz iin tablodan inceleyiniz.
